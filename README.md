# Gravitational Lensing Mass Mapping

Author: Aaria Joshi

This project explores gravitational lensing in galaxy clusters using astronomical data.

Goal:
Reconstruct the mass distribution of galaxy clusters by analyzing distortions in background galaxies.

Project Structure:

01_lensing_foundations.ipynb → theoretical background and project setup  
02_fits_data_exploration.ipynb → loading astronomical data  
03_arc_detection.ipynb → detecting gravitational lensing arcs  
04_mass_mapping.ipynb → reconstructing mass distribution
